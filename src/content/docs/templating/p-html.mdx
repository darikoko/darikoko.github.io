`p-html` sets the "innerHTML" property of an element to the result of a given expression.

<Aside type="caution">
Only use on trusted content and never on user-provided content.

Dynamically rendering HTML from third parties can easily lead to XSS vulnerabilities.
</Aside>

## Simple example

```html title="index.html"
<div p-html="'<strong>Author :</strong><span>Loïc Cavinato</span>'"></div>
```

<div class="result" set:html={`
<div p-html="'<strong>Author: </strong><span>Loïc Cavinato</span>'"></div>
    ` }>
</div>
